# Concurrent Banking System

Single-file C++ OS final project that demonstrates:

- CPU scheduling: FCFS, Priority, Round Robin
- Synchronization: mutex, condition variable, semaphore-style payroll control
- Deadlock avoidance: Banker's algorithm
- IPC: request/response style transaction flow
- Memory management: FIFO and LRU page replacement

## Project Files

- `task1.cpp` - main implementation and menu-driven simulator
- `REPORT.md` - generated summary of sample runs and metrics
- `project_inputs.txt` - sample input guide for each menu option
- `subhan_ahmad_resume.pdf` - resume copy included in the repository
- `Final Project_OS_4AB.pdf` - original project handout/reference
- `artifacts/` - charts, logs, and CSV outputs generated by the project

## Build

Use a C++17 compiler with pthread support.

```bash
g++ -std=c++17 task1.cpp -o task1 -pthread
```

On Windows with MinGW, the command is the same. If you use MSVC, compile with the Visual Studio C++ toolchain instead.

## Run

```bash
./task1
```

The program opens a menu for each module and can also generate the full report and all artifacts.

## Menu Modules

1. CPU scheduling
2. Synchronization
3. Banker's algorithm
4. IPC / message passing
5. Memory management
6. Run complete project and generate report + artifacts
7. Run built-in validation
